AES-256-CBC Key Facts
  • AES — Advanced Encryption Standard (NIST approved)
  • 256-bit key — 2²⁵⁶ possible keys; brute-force infeasible
  • CBC mode — each block XOR'd with previous ciphertext block
  • Random IV — prevents identical plaintexts from producing identical ciphertexts
  • PKCS#7 padding — pads input to 16-byte block boundary
  • Symmetric — same key for encrypt and decrypt (unlike RSA)
