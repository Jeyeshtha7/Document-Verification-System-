Key Properties of SHA-256
  • Deterministic — same input always gives same output
  • One-way (pre-image resistant) — cannot reverse hash to get input
  • Avalanche Effect — 1 bit change → ~50% of hash changes
  • Fixed size — any input → always 256-bit (64 hex chars)
  • Collision resistant — computationally infeasible to find two inputs with same hash
